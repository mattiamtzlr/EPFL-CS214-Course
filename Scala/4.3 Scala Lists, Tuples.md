#anonymous-functions 

Shorthand syntax for declaring [[2.1 Higher-Order Functions#Anonymous functions|anonymous functions]]:
```Scala
def foo(f: (Int, Int) => Int)(a: Int, b: Int): Int =
	f(a, b)

val add = foo(_ + _)
val mul = foo(_ * _)
```

```Scala
scala> add(3, 4)
val res0: Int = 7

scala> mul(3, 4)
val res1: Int = 12
```
